🏛️ Sistema de Gestión de Visitas Turísticas

Dirección de Turismo - Ciudad de Esperanza

🎉 Bienvenido al Sistema

🎯 Misión: Facilitar la gestión de visitas turísticas para instituciones educativas de nivel Primario y Secundario.
🏛️ Turismo de Identidad: Visitas para instituciones locales de Esperanza que buscan conocer su patrimonio cultural.
🎓 Turismo Educativo: Visitas para instituciones externas que desean conocer nuestra región con fines educativos.

ℹ️ Información del Sistema

📚 Niveles Educativos:

El sistema está configurado específicamente para manejar instituciones de nivel Primario y Secundario según las especificaciones de la Dirección de Turismo de Esperanza.

1. Registrar solicitud → 2. Seleccionar empresas → 3. Confirmar fechas → 4. Generar itinerario
